## Features

- Customable shortcut
- Keyboard events
- App search
- Google translate integration
- Calendar integration
- OpenWeather API integration
- AppleScript commands (sleep, lock, dark mode, etc.)
- Custom links/Custom AppleScripts
- Window Manager
